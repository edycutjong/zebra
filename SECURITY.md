# Security Policy

## Supported Versions

We actively support and patch the following versions of this project:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of our ZK proof schemes, smart contracts, and web applications very seriously.

If you believe you have found a security vulnerability in this project, please do not report it via a public GitHub issue. Instead, please report it privately by emailing us at **edy.cu@live.com**.

Please include as much detail as possible in your report, including:

- A description of the vulnerability and its potential impact.
- Step-by-step instructions or a proof-of-concept (PoC) to reproduce the issue.
- Any suggestions for mitigation or remediation.

Once received, we will acknowledge your report within 48 hours and work with you to coordinate a security fix and disclosure timeline.
